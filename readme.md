FileWiz is a web-based utility platform that helps users compress, extract, optimize, and manage files such as .zip, .pdf, .pptx, and .docx in one unified interface.
Instead of relying on multiple tools or installing heavy desktop software, FileWiz provides a fast, secure, and developer-friendly solution directly in the browser.


TechStack : 
Frontend
1. HTML
2. CSS
3. EJS (templating engine)

Backend
1. Node.js
2. Express.js

Processing & Tools
1. Python
2. Ghostscript
3. FFMPEG

Requirements : 
1. Python - For compression and optimisation logic.
2. GhostScript - For PDF Optimization and Compression.
3. FFMPEG - Media Compression 

RELIABLE features : 
1. No permanent file storage. 
2. Auto file cleanups.
3. Isolated processing for each request.

Current Features : 
1. PEEK .zip files and edit or create files in it without openning it. 
2. PDF Compressor provides Lossless compression of PDF.
3. PPT Compressor provides user with various options of compressing the ppt (Less, Normal, Extreme) and also gives user ability to remove all pictures form the files.
